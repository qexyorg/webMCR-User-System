Установка - Шаг 1

Название модуля
- Название модуля системы пользователей (Отображается в заголовке страницы и хлебных крошках)
Использовать комментарии
- Функция позволяет комментировать профили пользователей
Использовать MailBox
- Если вы используете модуль "MailBox", то при просмотре полной информации о пользователе, будет выводиться кнопка для отправки сообщения пользователю
