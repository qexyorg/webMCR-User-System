Название
- Название модуля системы пользователей (Отображается в заголовке страницы и хлебных крошках)
Использовать комментарии
- Функция позволяет комментировать профили пользователей
Пользователей на страницу
- Кол-во выводимых пользователей на одну страницу
Комментариев на страницу
- Кол-во выводимых комментариев на одну страницу
Уровень доступа к списку пользователей
- Минимальный уровень доступа к списку пользователей
Уровень доступа к панели управления
- Минимальный уровень доступа к панели управления статическими страницами
Использовать MailBox
- Если вы используете модуль "MailBox", то при просмотре полной информации о пользователе, будет выводиться кнопка для отправки сообщения пользователю
