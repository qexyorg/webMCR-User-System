Установка

Внимание! Данной установкой стоит пользоваться только в том случае, если ранее данный модуль не был установлен

В противном случае, воспользуйтесь обновлением


1. Заливаем все файлы и папки в главный каталог WebMCR с заменой текущих

2. Выставляем права на запись папки configs и файла us.cfg.php

3. Открываем файл .htaccess и после строчки

RewriteRule ^go/([^/]+)/?$ index.php?mode=$1 [L,NE]

- Добавляем

RewriteRule ^go/users/page-(\d+)/?$ index.php?mode=users&pid=$1 [L,NE]
RewriteRule ^go/users/(\w+)/?$ index.php?mode=users&do=$1 [L,NE]
RewriteRule ^go/users/full/(\w+)/?$ index.php?mode=users&do=full&op=$1 [L,NE]

4. Открываем instruments/menu_items.php и перед строчкой

'options' => array (

- Добавляем

'users' => array (

'name' => 'Пользователи',
'url' => Rewrite::GetURL('users'),
'parent_id' => -1,
'lvl' => 1,
'permission' => -1,
'active' => false,
'inner_html' => '',
),
